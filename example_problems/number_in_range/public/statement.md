You are given two integers, $l$ and $h$ ($l \leq h$). Your goal is to find the
smallest integer $n$ such that $l \leq n \leq h$.

## Input Format

The first line of input contains a single integer $l$ ($1 \leq l \leq 100$).

The second line of input contains a single integer $h$ ($1 \leq h \leq 100$).

For example:

```
1
5
```

## Output Format

The output should contain exactly one line containing only the value $n$.

For example, a valid output for the input above is:

```
4
```

Note that a better solution exists with $n = 1$. An output does not need to be
optimal to be valid.
